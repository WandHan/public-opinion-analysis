## 手动启动（高级用户）

### 方法1：Python HTTP服务器

```bash
# 进入项目目录
cd 精简分析报告_20251119

# 启动服务器
python -m http.server 8000

# 访问地址
http://localhost:8000
```

### 方法2：使用Node.js

```bash
# 安装http-server（首次）
npm install -g http-server

# 启动服务器
http-server -p 8000

# 访问地址
http://localhost:8000
```

## ❓ 常见问题

### Q1: 页面显示"正在加载数据..."不消失
**原因**：未使用HTTP服务器，浏览器阻止了fetch请求
**解决**：使用`启动服务器.bat`或手动启动HTTP服务器

### Q2: 数据显示为空
**原因**：JSON文件路径错误或格式不正确
**解决**：
1. 检查 `analysis_results/` 目录下是否有JSON文件
2. 使用JSON验证工具检查格式
3. 查看浏览器Console(F12)错误信息

### Q3: 部分数据显示异常
**原因**：JSON字段与预期不符
**解决**：参考`数据挂载说明.md`中的字段映射表
