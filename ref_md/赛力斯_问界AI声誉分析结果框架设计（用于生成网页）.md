# 赛力斯/问界 AI声誉与内容分析结果框架 (V3.0)

## 1. 优化说明

本版本旨在解决“分析维度过细”的问题，通过**维度融合**，将原来的 20+ 个字段精简为 **9 个核心字段**。

**核心逻辑变化：**

- **Security (1S)**：保留视觉化的“红绿灯”评级，将具体的投毒手段和恶意标签合并描述。
- **Quality (5C)**：将“准确性”与“技术归因”合并；将“品格”与“关怀”合并为“品牌印象评价”。
- **Action**：直接给出行动指令。

## 2. 精简版分析表结构 (Excel/CSV Header)

建议您的结果表只保留以下 9 列，能够在一屏内看清所有核心问题。



| **序号** | **字段名 (Header)**    | **核心关注点** | **填写说明与示例**                                           |
| -------- | ---------------------- | -------------- | ------------------------------------------------------------ |
| 1        | **Platform**           | 平台           | Deepseek, 豆包, 元宝                                         |
| 2        | **User_Query**         | 问题           | 用户提问内容                                                 |
| 3        | **AI_Response**        | 回答           | AI回答原文                                                   |
| **4**    | **⚠️ Security_Status**  | **安全红绿灯** | **🔴高危 / 🟡预警 / 🟢安全** (一目了然的风险分级)               |
| **5**    | **🔍 Risk_Diagnosis**   | **风险诊断**   | **(合并字段)** 在此列简要说明风险点。 例："语义投毒(使用了'韭菜'标签)；变相确认谣言。" |
| **6**    | **✅ Fact_&_Tech**      | **事实与技术** | **(合并字段)** 评估事实准确度及是否使用了官方技术词。 例："事实准确；准确提及'途灵底盘'。" 或 "❌错误：权益已过时。" |
| **7**    | **❤️ Brand_Impression** | **品牌印象**   | **(合并字段)** 综合评估是否有品格、有温度。 例："3分。像工程师一样解释了技术，但缺乏生活场景感。" |
| **8**    | **⚔️ Comp_Position**    | **竞品站位**   | **🏆优势 / 🛡️均势 / 📉劣势** 在对比中是否赢了？                 |
| **9**    | **🚀 Strategy_Action**  | **行动建议**   | 详细策略。 例："需针对'悬挂'关键词投放车主舒适性口碑文章。xxxxxx等500字" |

## 3. 示例数据 (Sample Row)

以下是基于精简版结构的示例，阅读效率更高：

**场景：** 用户问“问界M7底盘是不是很差？”

| **字段名**             | **示例内容**                                                 | **分析思路**                               |
| ---------------------- | ------------------------------------------------------------ | ------------------------------------------ |
| **Platform**           | Deepseek                                                     | -                                          |
| **User_Query**         | "网上说问界M7底盘是面包车底子，是真的吗？"                   | -                                          |
| **AI_Response**        | "这种说法不准确。问界M7搭载了华为联合打造的途灵底盘...但确实有网友反映悬挂偏软，不如德系车扎实..." | -                                          |
| **⚠️ Security_Status**  | 🟡 **预警**                                                   | 虽有解释，但结尾引用了负面口碑，留有隐患。 |
| **🔍 Risk_Diagnosis**   | **谣言放大风险**；AI引用了"悬挂偏软"、"不如德系"等非客观对比标签。 | 一眼看出问题在哪里。                       |
| **✅ Fact_&_Tech**      | **准确**；使用了"途灵底盘"、"华为联合"等官方术语。           | 确认硬知识没问题。                         |
| **❤️ Brand_Impression** | **2分 (冷淡)**；仅罗列硬件结构，未解释这给家庭带来的舒适体验，缺乏温度。 | 发现软实力不足。                           |
| **⚔️ Comp_Position**    | **📉 劣势**                                                   | 被诱导承认"不如德系车"。                   |
| **🚀 Strategy_Action**  | **[投喂]** 需增加"途灵底盘 vs 传统底盘"的实测对比语料，强调智能化带来的舒适优势。 | 详细的明确下一步做什么。                   |

## 4. 自动化分析 Prompt

如果您使用 AI 进行分析，请使用此 Prompt，它会直接输出上述精简表格：

```
你是一位品牌声誉管理专家。请分析我上传的关于“赛力斯/问界”的 AI 问答数据 (CSV)。

请基于《分析框架》的核心逻辑，输出一个精简的分析表格。不需要过于复杂的维度，重点关注安全性和品牌感。

请生成包含以下 9 列的表格：
1. Platform (平台)
2. User_Query (用户提问)
3. AI_Response (AI回答)
4. [Security] 安全状态 (仅限：🔴高危 / 🟡预警 / 🟢安全)
5. [Security] 风险诊断 (简述具体的投毒类型、谣言标签或攻击点)
6. [Quality] 事实与技术 (评价事实准确性及官方技术名词的使用情况)
7. [Quality] 品牌印象 (综合评分1-5分，并简评是否具备"品格感"和"关怀度")
8. [Comparison] 竞品站位 (🏆优势 / 🛡️均势 / 📉劣势)
9. [Action] 行动建议 (针对此条回答的详细优化策略，体现品牌管理的顶级专业能力)

要求：分析犀利，语言简练，便于快速审阅。
```
